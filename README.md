# Synth City: Harmonic Horizons

A multiplayer musical city-builder where every building is an instrument. Collaborate with others to create a global symphony and unlock new sounds through progression.

## 🚀 Features

### 🎵 Musical Architecture
- **Oscillator Buildings**: Continuous tones with customizable waveforms.
- **Sequencer Buildings**: Programmable 4-step patterns.
- **Sampler Buildings**: Rhythmic noise loops.
- **Arpeggiator Buildings**: (New) Automatic melodic patterns.
- **FX Buildings**: (New) Local audio processing.
- **Global FX**: A unique building that affects the entire city's soundscape (Reverb/Delay).

### 🏗️ City Building & Infrastructure
- **Roads**: Connect buildings to synchronize their timing and create complex loops.
- **Grid System**: Precise placement with snapping and visual feedback.
- **Undo/Redo**: Full history support for building operations.

### 📈 Progression & RPG Elements
- **XP & Leveling**: Earn XP by building and maintaining harmony.
- **Level Unlocks**: New building types unlock as you level up.
- **Upgrades**: Enhance buildings to generate more Harmony Points (HP) or unlock advanced synthesis parameters.

### 🎨 Visuals & UX
- **Real-time Visualizer**: The city pulses and glows in sync with the musical output.
- **Selection HUD**: Detailed configuration panel for every building.
- **Multiplayer**: A shared world where you see and hear everyone's creations.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Graphics**: Konva.js (Canvas API)
- **Audio**: Tone.js
- **Backend**: Firebase (Firestore & Auth)
- **Animation**: Framer Motion

## 🎹 Controls
- **Select Tool**: Click buildings to configure them.
- **Place Tool**: Add new structures to the grid.
- **Delete Tool**: Remove your own buildings.
- **Road Tool**: Draw connections between buildings.
- **Zoom/Pan**: Use mouse wheel and drag to navigate the grid.
